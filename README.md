## How to Add This to Your Profile

You can easily add these stats to your **GitHub Profile README**.

### Step 1: Fork & Setup
1. **Fork this repository** to your own account.
2. Go to the **Actions** tab in your forked repo and click **"I understand my workflows, go ahead and enable them"**.
3. (Optional) Manually trigger the **"Update Streak Stats"** workflow once to generate your first set of images immediately.

### Step 2: Add to Your Profile
Copy the code below and paste it into your profile's `README.md`.
**Important:** Replace `YOUR_USERNAME` with your actual GitHub username.
